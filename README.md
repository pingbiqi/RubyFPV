## ## 🌟 为什么选择 RubyFPV？

在数字图传领域，RubyFPV 以其独特的**底层链路优化**脱颖而出。相比于传统模拟图传或其他开源方案，它具备以下降维打击的优势：

### **1. 极强的抗干扰能力 (ARQ & Adaptive Bitrate)**
* **自动重传机制**: 即使在复杂的城市电磁环境下，丢包也能即时找回，画面不闪烁。
* **自适应码率**: 实时感应链路质量，动态调整带宽，确保“图传不断、操控不丢”。

### **2. 业内领先的分集接收 (Advanced Diversity)**
* 支持多达 **4 路以上**的无线网卡同时工作。
* 完美兼容 RTL8812AU、RTL8811CU 等高性能 5.8G 芯片，大幅提升信号覆盖广度。

### **3. 深度定制的 OSD 系统**
* 毫秒级响应的实时遥测数据。
* 支持多种飞控协议（Mavlink, LTM 等），所有飞行数据一目了然。

### **4. 硬件生态灵活**
* **天空端**: 支持树莓派 Zero/3/4 及 OpenIPC 模组。
* **地面端**: 兼容各种大功率 USB 网卡，配合定向天线可实现 **20km+** 极限拉距。

## ## 🔧 硬件推荐清单 (炳光无线实测)

为了发挥 RubyFPV 的最大性能，建议搭配以下硬件组合：

| 硬件类型 | 推荐型号 | 优势说明 |
| :--- | :--- | :--- |
| **主控板** | Raspberry Pi Zero 2W / OpenIPC | 体积小、发热低、性能强 |
| **无线网卡** | ASUS USB-AC56 (RTL8812AU) | 业内公认的链路稳、功率足 |
| **天线方案** | 枫叶/华硕 定向平板 + 蘑菇头 | 远航必备的高增益组合 |
| **增益模块** | 2.4G/5.8G 双向功放 (PA) | 针对超长距离飞行任务定制 |
